# **App Name**: Miso's Wellness Hub

## Core Features:

- Personalized Daily Workout Checklists: Dynamically display and track completion for both standard and a 'low energy day' workout checklists, tailored to user's activity level.
- Local Data Persistence & Multi-Theme Support: Securely save all user progress, settings, accumulated points, and theme preferences directly to the browser's local storage. Includes an option to switch between 'Purple', 'Pink', and 'Dark' visual themes.
- Gamified Point System: Users earn points based on completion of tasks and specific daily/streak bonuses: Warm up (+5), Cardio full (+20), Cardio low energy (+10), Strength (+15), Arms (+10), Stretch (+5), Steps goal hit (+20), Water goal (+10), Nourishing food (+10), Completed full workout (+30), Low energy day completed (+15), 3-day streak bonus (+25), and 7-day streak bonus (+75).
- Interactive Point Shop & Titles System: Allows users to spend earned points on specific rewards (e.g., 'One more game' for 100 points, 'Silly selfie' for 300 points, 'Nap Pass' for 500 points) and unlock/display various titles ('Consistency Queen' for 250 points, 'Aura Farmer' for 1000 points, etc.).
- Interactive Water Tracking & Daily Workflow: A dedicated component for logging daily water intake, featuring a unit toggle (liters/ounces) and interactive input methods. Includes a daily task reset mechanism and a 'day streak' counter.
- Celebratory UI Feedback: Implement engaging visual feedback such as confetti animations and encouraging messages ('Good job my pretty girl!') upon key achievements like finishing a daily workout or purchasing a reward.
- Progressive Web App (PWA) Integration: Enable 'Add to Home Screen' functionality, allowing the web application to be installed and launched directly from a mobile device's home screen, behaving like a native app.

## Style Guidelines:

- Primary color: A soft, muted lavender ('#B899DB') to evoke a sense of calm and encouragement, suitable for a light default theme.
- Background color: A very pale violet-white ('#F7F1FF'), offering a clean, non-distracting canvas that pairs gently with the primary color.
- Accent color: A vibrant yet cute pink ('#FF99FF') to provide visual pops and highlight interactive elements, harmonizing with the requested 'pink' theme option.
- Body and headline font: 'Poppins' (sans-serif) for its clean, contemporary, and geometric appeal, ensuring readability and a modern, cute aesthetic across all text.
- Utilize expressive and colorful icons or emojis for each section, goal, and interactive element (e.g., 💜, 🔥, 🏋️‍♀️, 💪, 🌿, 💧, 🍓, 🪙) to enhance visual appeal and communication within the card layout.
- The layout features a clean, card-based design with prominent rounded corners for each checklist section and UI element. A personalized header will display a title and a motivational message. The overall design prioritizes a clutter-free, responsive, and mobile-friendly presentation.
- Subtle animations for user interactions (e.g., checklist item completion, point acquisition), combined with a special confetti effect and encouraging messages for major milestones. The background will feature a soft, slowly moving gradient, unique to each selected theme (Purple, Pink, Dark mode).
